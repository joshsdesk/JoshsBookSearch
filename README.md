## ğŸ”§ Installation

1. **Clone the repo**  
   ```bash
   git clone https://github.com/joshsdesk/JoshsBookSearch.git
   ```

2. **Install dependencies**  
   ```bash
   cd server
   npm install

   cd ../client
   npm install
   ```

3. **Set up environment variables**  
   In `server/.env`:
   ```
   MONGODB_URI=<your MongoDB Atlas URI>
   JWT_SECRET_KEY=<your secret>
   ```

4. **Build and run the server**  
   ```bash
   cd server
   npm run build
   npm run start
   ```

5. **Start the client**  
   ```bash
   cd ../client
   npm run dev
   ```
